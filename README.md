# Finalproject# SoleMate - Premium Shoe Store

A fully functional e-commerce website for selling premium footwear, built with PHP and MySQL.

## Features

### Frontend
- Responsive design (mobile, tablet, desktop)
- Product catalog with filtering and sorting
- Product search functionality
- Shopping cart with quantity management
- Secure checkout process
- User registration and login
- User dashboard (order history, wishlist, profile)
- Interactive size guide with calculator
- Store locator with interactive map

### Admin Panel
- Dashboard with sales analytics
- Product management (CRUD operations)
- Order management with status updates
- User management (disable/enable accounts)
- Category management
- Template switcher (3 themes: Light, Dark, Autumn)
- System monitoring page

### Technical Features
- SEO-friendly URLs (mod_rewrite)
- Session-based shopping cart
- Persistent cart for logged-in users
- Secure password hashing (bcrypt)
- SQL injection prevention (PDO prepared statements)
- XSS protection
- CSRF protection

## Technology Stack

| Layer | Technology |
|-------|------------|
| Backend | PHP 7.4+ |
| Database | MySQL 5.7+ |
| Frontend | HTML5, CSS3, JavaScript |
| CSS Framework | Custom CSS with responsive design |
| Charts | Chart.js |
| Maps | Leaflet.js |
| Icons | Font Awesome 6 |
| Fonts | Google Fonts (Inter) |
